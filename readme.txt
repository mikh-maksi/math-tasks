По задачах:
1. Описати виведення фігур під задачі. 
2. Продумати слой із координатами: коли на вході параметри задачі, на виході - координати фігури. А вже фігура відмальовується типовою функцією.
3. Позначити ті проблеми в відмалювуванні фігур, які існіють на даний момент.
4. Сформуваи план усунення проблем, що існують для відмальовки.
5. Згенерувати задачі. Продумати структуру тексту та генератору параметрів.
